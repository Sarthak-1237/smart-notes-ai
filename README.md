
# 📚 AI Smart Notes & Lecture Summarizer

## Objective
A rapid AI tool built to convert long lectures and PDF documents into short summaries, key extracted points, and revision flashcards instantly. 

## Expected Outcomes Achieved
✅ AI-powered note summarizer
✅ Automatic key point extraction
✅ Quiz or flashcard generation from notes

## Tech Stack
* **Frontend:** Streamlit (Python)
* **AI Engine:** Google Gemini 1.5 Flash API
* **Document Processing:** PyPDF2

## How it works
1. The user uploads any PDF lecture slide or document.
2. The system extracts the raw text.
3. The text is processed through the Gemini AI model with a specialized tutoring prompt.
4. The user receives an instant, formatted study guide.
